Steps Required to Develop on Server

Author: SandZn Date: June 12, 2022  21:33:30

Log in with your KTH account

Before logging in to the server website, you need to connect it with your KTH account: username@ug.kth.se, then the account will jump to the KTH log-in page.

Create an owner for you

https://portal.azure.com/#home

Create an owner.

Add resources to the TCS group

  1. Find the TCS group
  2. Create a virtual machine step by step. Generally, you carefully choose the size (or edit the size later), for the other options or input, just fill in the default value till the final step.
  3. Deal with your ssh key.
